# Boolean Expressions
**True** or **False**

```
bool1 = True
bool2 = 3 * 3 == 9
bool3 = False
bool4 = 3 * 3 != 9

print(bool1, bool2, bool3, bool4)
print(type(bool1), type(bool2), type(bool3), type(bool4))
```

**Note:** `True` is a boolean variable, but ``"True"`` is a string (note the quotation marks)
