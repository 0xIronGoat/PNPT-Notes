# Maths
##### Basic maths:
```
print(50 + 50) # = 100
print(40 - 10) # = 30
print(3 * 15) # = 45
print(40 / 10) # = 4
```

##### Follows standard order of operations (i.e. PEMDAS or BODMAS):
```
print(50 + 40 * 2 / 10) # = 58
print((50+40) * 2 / 10) # = 18
```

##### Exponents:
```
print(3 ** 2) # = 9
```

##### Modulo (divide and return the *remainder*):
```
print(50 % 6) # = 2
print(50 % 5) # = 0
```

##### Division:
```
print(50 / 6) # = 8.333333333333334
print(50 // 6) # = 8 (only gives the whole number)
```
