# Subnetting

```
| 255      . 255      . 255      . 0        |
| 11111111 . 11111111 . 11111111 . 00000000 |
```



### Subnet Mask vs CIDR Notation
```
| Subnet Mask     | CIDR        |
| --------------- | ----------- |
| 255.0.0.0       | /8          |
| 255.255.0.0     | /16         |
| 255.255.255.0   | /24         |
```
